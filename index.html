Projects

OSPF-ABR / Arbitrary Topologies

MSc thesis · 2017–2018 · Python

An extension of the OSPF routing protocol that lifts the classic two-level area hierarchy and supports arbitrary topologies between Area Border Routers. It introduces new inter-area communication channels so the protocol can compute optimal end-to-end paths across any area layout, while staying backwards-compatible with stock OSPF deployments.

Built and validated as part of my MSc dissertation at Instituto Superior Técnico, supported by Instituto de Telecomunicações.
